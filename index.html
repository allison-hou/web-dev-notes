Web Development 網站架構 & 設計

HTML

整體結構 :

<!doctypehtml> 
<html>
    <head>
        <title>網站的名稱</title>
    </head>
    <body>
    所有內文
    </body>
</html>

常用的tags :

<p>...</p>     paragraph 段落
<h>...</h>     heading 標題
<em>...</em>   italic 斜體
<b>...</b>     bold 粗體
<br>           line break 換行
<ul>...</ul>   unorder list 無排序清單
<ol>...</ol>   order list 排序清單
註 : <p>和<br>不一樣，前者上下個空一行，後者只有換行

常用語法

  • How do we add links ?

Use my video online learning as an example :

<a href="https://www.youtube.com/watch?v=3JluqTojuME">online learning</a>
  • How do we add images ?

For example :

<img src="">...</img>
  • How to make an unorder list ?

For example :

<ul>
    <li>item1</li>
    <li>item2</li>
</ul>

By the way, if we want a order list, just change the <ul> tags to <ol> instead.

CSS

整體結構 :

<style type="text/css">
    body{

    }
</style>

引用CSS

  <style href="styles/main.css rel="stylesheet" type="text/css">

顏色的幾種寫法

  1. 直接寫顏色

    ex. tomato

  2. 色號

    ex. #333

  3. RGB

    ex. rgb(255, 12, 0)

  4. HEX

    ex. #ff6845

  5. HSL

    ex. hsl(9, 30%, 74%)

  6. rgba

    ex. rgba(235, 39, 54, 0.5)

  7. hsla

    ex. hsla(5, 45%, 39%, 0.5)

字元網底

background-color: #999; (ex.#999灰色 #555淺灰 #000黑色)

字的顏色

color: white;

邊框

border: 2px solid Violet;

字體

font-family: arial;

水平對齊

text-align: center;

內距設定(長/寬)

padding: 10px 10px 1px 1px;

背景

  • opacity : 用於背景顏色透明程度(由 0.0 到 1.0)

  • background-image : 放背景圖片

    語法 : url(圖片地址)

  • background-repeat: repeat-x 重複展示照片

    (x代表水平，y代表垂直)

  • background-repeat: no-repeat 照片只出現一次

Selectors

. 代表 class
# 代表 id 
註 :
1.id和class的差別在於id一個網頁只能有一個，class則無限制 !
2.class and id name cannot start with a number
* 代表全部元素

註解

CSS :
/* 一段文字 */

<!-- 一段文字 -->
